Wind Visualization

A Particle simulation that is based directly on data from a Wind Tunnel analyzing the wind coming off of a square-back car. The tool has a ton of customization options and runs in real-time using Unity, with additional support for VR using MRTK toolkit, as well as midair haptics using an ultrahaptics board (not shown).
